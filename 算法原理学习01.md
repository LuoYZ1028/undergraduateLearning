# 总体流程

- 文本检测(和**目标检测**类似)：哪里有文字，文字的范围是多少
- 文本识别：对定位好的文本区域进行识别，主要解决每个文字是什么，将图像中的文字区域转化为字符信息

因此文字识别的流程大致是：
```mermaid
graph LR;
输入-->文字检测-->文字识别-->输出;
```
